mpy Language

mpy Programs are written using the mpy Language.

  • mpy Structure, the mpy language follows the same basic structure as Python. Just think of mpy as a small subset of Python.
  • mpy Functions, the mpy language supports several commands to directly control the microcontroller.
  • mpy Registers, the mpy language uses the full range of MSP430 register definitions available within the mspgcc compiler. What this means is that you can access any of the internal registers within the MSP430 to give complete control of the hardware, even if there is no easy to use mpy function available.

Example mpy Program snippets are provided for many of the commands. Most are fully working programs that can be run directly on the microcontroller. The example text can be cut and pasted into the mpy Editor and run directly into the microcontroller.
