adc_enable( )

Syntax: adc_enable( )
Description: Enables the Analog to Digital Converter circuit.
Parameters: None
Returns: Nothing
Related: adc()

Example:

adc_enable()                           # Enable the ADC block
while 1:
   val = adc( P1_4 )                   # Use the ADC to convert voltage at ppin P1_4 into a 10 bit number
  Â print_value( 'ADC P1_4 is:', val )
   wait( 1000 )
