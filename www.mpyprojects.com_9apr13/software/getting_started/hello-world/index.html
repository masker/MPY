3 Hello World

It’s something of a tradition when learning a new computer language to start with a program that prints out Hello World. The reason being it’s probably the simplest program that you can write that actually does something.

Well here it is in mpy:

print('Hello World')

It couldn’t be simpler!  Go ahead type it into the  mpyEditor and hit the Prog button. (make sure it is the only line in your program)

print

The print function is used to print onto the MpyUart window. If you have setup the mpyEditor correctly you should see Hello World appear in the MpyUart window. That is the window on the bottom right side of the mpyEditor (If you don’t see it look here for some help).

Note: You may see the first characters on the line are messed up.  Don’t worry about this, it’s a glitch on the first print in the program.

You can print text or variables with the print command,  and you can use the print command to print multiple values.

Text to be printed must be enclosed in quotes like ‘text’, (or you can use double quotes “text”)

Try

a = 4
print( 'Oh No! There are ', a , 'creepers' )
print( 'sssss BOOM')

This will print out

Oh No! there are 4 creepers
sssss BOOM

debug

In addition to the print command there is the debug command. The debug command is very similar to the print command except that it prints  out in a different format, it shows the function name and the line number of the debug statement  and it also prints out the variable name.

a = 5
debug( a )

Gives:

(main:2)  a=5

Using the debug command in your program will make it easier to see what is happening inside your program when it is running, and it will help you track down the bugs and get your program running correctly.
